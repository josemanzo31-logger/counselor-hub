# Counselor Hub 🚀

Welcome to the **Counselor Hub**! This repository serves as a centralized digital workspace for student support resources, instructional design templates, and counseling tools designed to enhance student success and engagement.

## 📁 What's Inside?

* **`TransferPathwayExplorer_2.jsx`** 🎓: An interactive, dynamic React component built to help students seamlessly explore, map out, and navigate their academic transfer pathways.
* **`personal-finance-fundamentals_1.html`** 📈: A clean, accessible web resource dedicated to teaching students the foundational pillars of personal finance, budgeting, and long-term financial literacy.

## 🛠️ Tech Stack & Tools
* **Frontend:** React (JSX), HTML5, CSS3
* **Focus Areas:** Digital Pedagogy, Student Equity, Educational Technology, and Financial Literacy.
